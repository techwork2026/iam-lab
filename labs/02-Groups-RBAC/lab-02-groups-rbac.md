# Lab 02 — Groups and RBAC

## Objective
Use Microsoft Entra ID groups and role assignments to simulate group-based access control.

## Scenario
A company organizes users into departments and assigns permissions based on job function.

Groups to create:
- IT-Team
- Finance-Team
- HR-Team

Users to assign:
- Samantham
- ContractorJason
- MaryIntern

## Tasks
- Create security groups
- Add users to groups
- Assign role to group
- Verify role assignment

## Skills Demonstrated
- Microsoft Entra ID
- Security groups
- RBAC
- Group-based access control
